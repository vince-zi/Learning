# Layer 3: MODEL - 系统抽象模型

## 1. 知识图谱与数据模型 (Knowledge Graph & Entity Schema)
* 定义系统中的核心实体（如：用户、会话、任务、成果）以及它们之间的关联。

## 2. 状态机模型 (State Machine)
* 系统在交互过程中，对象存在哪些状态？它们如何通过用户的行为或时间发生转移？

## 3. 决策树 / 算法逻辑 (Decision Tree)
* 系统根据什么具体的条件分流策略？（例如，AI 在何时判定用户完成了第一阶段，转入第二阶段？）

## 4. 反馈机制 (Feedback Mechanism)
* 系统如何感知用户的动作？
* 如何给予即时且正向的反馈，促进用户的下一次行为？
